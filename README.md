# HustleBoard ZA — Local Gig Marketplace

A modern, responsive gig marketplace for South Africa.

## Live Demo
You can view the live site here: [https://dumbellienumph-design.github.io/-HustleBoard-ZA/](https://dumbellienumph-design.github.io/-HustleBoard-ZA/)

## Features
- Clean, modern dark UI
- Category filtering
- Responsive design
- South African focused categories (Home Services, Tech, Creative, etc.)

## Deployment
This project is deployed using GitHub Pages.
